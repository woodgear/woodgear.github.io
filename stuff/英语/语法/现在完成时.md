# 定义
have/has + 动词过去式
现在完成时强调过去的事情对现在的影响
按照<<什么是句法>>15章 什么是时什么是体 上的定义  
现在完成时是已然体在英语中体现 具体手段就是特殊词+动词屈折变化(have/has +done)
# 识别
have/has + done
